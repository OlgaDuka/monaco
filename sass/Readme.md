Файлы для сборки стилей<br>

```bash
blocks/           # блоки
global/            
  fonts.scss      # шрифты проекта
  global.scss     # глобальные стили
  mixins.scss     # миксины
  variables.scss  # переменные
plugins/*.scss    # стили подключаемых плагинов
  
main.scss         # подключение всех стилевых файлов
```
